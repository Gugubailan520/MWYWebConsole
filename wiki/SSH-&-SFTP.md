# SSH 终端 & SFTP 文件管理

---

## SSH 终端

### 连接方式

#### 方式一：从主页连接

1. 在主页添加服务器，类型选择 **SSH**
2. 填写主机地址、端口（默认 22）、用户名和密码/密钥
3. 点击服务器卡片上的 **终端** 按钮即可打开

#### 方式二：快捷连接栏

在顶部输入框中输入 `user@host:port` 格式，回车直接连接：

```
root@192.168.1.1:22
admin@10.0.0.5
```

#### 方式三：embed.html 嵌入直连

```
/embed.html?type=ssh&host=192.168.1.1&port=22&user=root&pass=123456
```

详见 [iframe 嵌入](Embed)。

---

### 认证方式

#### 密码认证

在连接信息中填写密码字段即可。

#### SSH 私钥认证

在服务器配置中粘贴私钥内容（支持 OpenSSH 格式 PEM 格式），留空密码字段，系统自动使用密钥认证。

**embed.html 传递私钥**：

```
/embed.html?type=ssh&host=host&user=root&key=-----BEGIN OPENSSH PRIVATE KEY-----...
```

> 建议通过后端动态生成嵌入链接，避免在 URL 中暴露私钥。

### 终端功能

| 功能 | 说明 |
|------|------|
| 全功能终端 | 基于 xterm.js，支持 256 色、Unicode、鼠标 |
| 自适应大小 | 窗口缩放时自动发送 `SIGWINCH`，终端尺寸实时同步 |
| 终端透明度 | 设置面板可调整 0~100% 背景透明度 |
| 文件拖拽 | 拖拽文件到终端区域，触发 SFTP 上传 |
| 目录跟随 | 开启后，`cd` 命令自动同步 SFTP 文件浏览位置 |

---

### 设置面板

点击右上角 **⚙ 设置** 按钮打开：

| 设置项 | 说明 |
|--------|------|
| 目录跟随 | SFTP 面板自动跟随 `cd` 命令切换目录 |
| 显示隐藏文件 | SFTP 中是否显示 `.` 开头的文件 |
| 自动打开 SFTP | 连接建立后自动展开 SFTP 文件管理侧栏 |
| 自动打开监控 | 连接建立后自动展开服务器状态监控面板 |
| 终端透明度 | 调整终端背景的透明程度（0% 完全透明） |

## SFTP 文件管理

SFTP 侧栏与 SSH 终端共用同一连接，无需额外配置。

### 功能列表

| 功能 | 操作 |
|------|------|
| 浏览目录 | 点击目录名进入，顶部面包屑导航 |
| 上传文件 | 点击上传按钮或拖拽文件到文件列表区域 |
| 下载文件 | 点击文件行末尾的下载图标 |
| 在线编辑 | 点击文本文件，弹出在线编辑器，保存后直接写回服务器 |
| 删除文件 | 点击删除图标，弹窗确认 |
| 创建目录 | 点击新建目录按钮 |
| 显示隐藏文件 | 通过设置面板开关 |

### 目录跟随

开启「目录跟随」设置后，终端中执行 `cd /var/log` 时，SFTP 侧栏会自动切换到 `/var/log` 目录。

实现原理：监听终端输出，检测 shell 提示符变化时获取当前目录（`pwd`），更新 SFTP 路径。

## 独立 SFTP 页面

也可直接使用 `/sftp.html` 页面（不带终端的纯文件管理）：

```
/embed.html?type=sftp&host=192.168.1.1&user=root&pass=123456
```
